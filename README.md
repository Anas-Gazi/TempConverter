A simple C++ console program to convert temperatures between Celsius and Fahrenheit

## Features

1. Convert Celsius to Fahrenheit
2. Convert Fahrenheit to Celsius
3. Convert Fahrenheit to Celsius
4. Keep running until the user chooses to exit

## In this project, I learned:

1. How to create a simple temperature converter using cin and cout
2. How to use conditional statements
3. How to use a while loop to allow multiple conversions until exit
also 
1. How to initialize a Git repository, commit changes, and push code to GitHub.
2.How to manage versions of my project using Git
